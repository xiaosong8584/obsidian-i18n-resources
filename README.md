# 🌐 社区插件 & 主题 AI 汉化指南

欢迎下载obsidian i18n的翻译文件！

### 💡 适用范围 
本仓库基于 i18n plugin v2.0.0, 尽可能将安装和使用中的插件/主题 汉化。


### 💡 如何选择最合适的翻译版本？

在面对多个下载选项时，请遵循**“先看译文版本，再看插件版本”**的原则。

|**译文版本号**|**翻译方式**|**质量说明**|
|---|---|---|
|`1.0.0`|🤖 AI 自动提取翻译|未经人工校对，可能存在机翻感或翻译不全的问题|
|`2.0.0`|👤 人工校对|**经过人工精校，翻译更全面、表达更自然**|

### Note:
- 自动翻译环境搞通之前, 先人工翻译, 默认v1.0.1, 先用起来. 
- 翻译和原始插件的构架有关, 有的插件自带多国语言支持, 不必翻译; 有的插件无法提取/翻译; 后续统一整理一下。
